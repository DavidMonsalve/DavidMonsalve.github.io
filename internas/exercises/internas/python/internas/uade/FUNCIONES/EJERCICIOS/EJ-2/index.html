Exercise 2

Español:


2. Desarrollar una función que reciba tres números enteros positivos y verifique si corresponden a una fecha válida (día, mes, año). Devolver True o False según la fecha sea correcta o no. Realizar también un programa para verificar el comporta- miento de la función.


English:


2. Develop a function that takes three positive integers and checks if correspond to a valid date (day, month, year). Return True or False depending on the date is correct or not. Also carry out a program to verify the behavior lie of the function.


                    
                    def verificar(d, m, y):
                        Booleana = False
                        if (d > 0) and (d <= 31):
                            if (m > 0) and (m <= 12):
                                if (y >= 0) and (y < 2023):
                                    Booleana = True
                        return Booleana


                    dddd = int(input("Dia: "))
                    mmmm = int(input("Mes: "))
                    yyyy = int(input("año: "))

                    x = verificar(dddd, mmmm, yyyy)

                    print(x)
