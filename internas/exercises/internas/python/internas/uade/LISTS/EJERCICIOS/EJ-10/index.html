Exercise 10

Español:


10.Generar una lista con números al azar entre 1 y 100 y crear una nueva lista con los elementos de la primera que sean impares. El proceso deberá realizarse utili- zando listas por comprensión. Imprimir las dos listas por pantalla.


English:


10.Generate a list with random numbers between 1 and 100 and create a new list with the elements of the first that are odd. The process must be carried out using using list comprehension. Print the two lists on the screen.


                    from random import randint
                    def generar(l):
                        for i in range (10):
                            l.append(randint(1, 100))
                        return l
                    
                    vacia = []
                    y = []
                    x = generar(vacia)
                    
                    i = 0
                    while (i < len(x)):
                        if (x[i] % 2 != 0):
                            y.append(x[i])
                        i +=1
                    
                    print("original: ", x)
                    print("impares: ", y)
