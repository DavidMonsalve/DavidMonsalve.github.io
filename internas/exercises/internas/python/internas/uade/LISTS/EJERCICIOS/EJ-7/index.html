Exercise 7

Español:


7. Intercalar los elementos de una lista entre los elementos de otra. La intercalación deberá realizarse exclusivamente mediante la técnica de rebanadas y no se creará una lista nueva sino que se modificará la primera. Por ejemplo, si Iistal = [8, 1, 3] y Iista2 = [5, 9, 7], Iistal deberá quedar como [8, 5, 1, 9, 3, 7].


English:


7. Insert the elements of a list between the elements of another. collation shall be made exclusively by the slice technique and shall not create a new list but the first one will be modified. For example, if Iistal = [8, 1, 3] and Iist2 = [5, 9, 7], Iist should be left as [8, 5, 1, 9, 3, 7].


                    def intercalar(l1, l2):
                    i = 1
                    j=0
                    while (i < len(l1)):
                        l1.insert(i, l2[j])
                        i += 2
                        j += 1
                    l1.insert(i, l2[j])
                    return l1
                
                
                x = [8, 1, 3]
                z = [5, 9, 7]
                
                print(intercalar(x, z))
